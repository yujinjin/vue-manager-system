# vue-manager-system
后台管理系统
### 封装的独立组件列表
1.图片上传（img-upload）
2.视频上传（video-upload）
3.视频播放（web-player）
4.富文本框（web-editor）
5.数据图表（data-chart）
6.数据列表栏显示隐藏状态的切换（toggle-table-column）

### 封装的页面级组件列表
1.查询栏（search-form）
2.操作栏（action-bar）
3.数据分页列表（table-data)
4.弹窗表单（dialog-form）
5.页面普通表单（page-simple-form）
6.页面复杂表单（page-complex-form）
7.页面特殊表单（page-special-form）
